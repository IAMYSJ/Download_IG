# Download_IG
備份自己的 Instagram 照片

## 開始使用
### 環境
python 3.x 以上
### 執行
```
$ python downloadIg.py
```
```
$ Please enter IGurl(Enter q to leave): 
```
這裡要輸入自己的 Instagram 帳號 url
### url 範例
https://www.instagram.com/xxxxxx/

xxxxxx 是 Instagram 帳號
### 結果
會產生一個 Image 資料夾，裡面匯存放下載的照片
